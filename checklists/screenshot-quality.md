# 截图质量检查清单

## 文件检查

- [ ] 所有截图路径存在；
- [ ] 所有截图使用相对路径；
- [ ] 未使用短效 Figma 图片 URL；
- [ ] 文件名稳定、可读、全小写、连字符。

## 清晰度检查

- [ ] 宽度大于 1440px 的节点已补局部截图；
- [ ] 高度大于 900px 的节点已补局部截图；
- [ ] 宽高比大于 4:1 的截图已补局部截图；
- [ ] 高宽比大于 3:1 的截图已补局部截图；
- [ ] 登录页、表单页、详情页、看板页有清晰局部截图；
- [ ] Dialog、Table、Navigation、Alert、Message 等关键组件有清晰截图。

## 数量检查

- [ ] 普通组件截图不超过 1 张；
- [ ] 重要组件截图不超过 1 张总览 + 3 张局部；
- [ ] 页面 example 截图不超过 1 张总览 + 4 张局部；
- [ ] `design.md` 直接引用截图不超过 80 张。
